# 🏴‍☠️ 콘솔 보물찾기 게임 (C / Windows)

이 프로그램은 **Windows 콘솔 기반 보물찾기 게임**입니다.  
플레이어는 방향키를 이용해 보물을 찾아 이동하며,  
문제를 풀어야 보물을 획득할 수 있고, 지뢰를 밟으면 초기 위치로 돌아갑니다.  
또한 게임 완료 후 **플레이 시간 기록 저장 및 조회 기능(랭킹 정렬 포함)**을 제공합니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🎮 캐릭터 이동 | 방향키(↑↓←→)로 콘솔 맵 이동 |
| 💎 보물 찾기 | 맵에 3개의 보물이 숨겨져 있음 |
| ❓ 문제 출제 | 보물 획득 전 랜덤 연산 문제 출제 |
| 💥 지뢰 존재 | 지뢰(※)를 밟으면 시작 위치로 초기화 |
| ⏱ 기록 측정 | `clock()` 기반 플레이 시간 저장 |
| 🏆 랭킹 기능 | 플레이어 이니셜 + 기록 저장 및 정렬 출력 |
| 🔁 재시작 기능 | r 키 입력 시 즉시 게임 재시작 |
| 🎨 콘솔 색상 지원 | `SetConsoleTextAttribute()` 사용 |

## ▶️ 실행 방법

### ✅ 1. 컴파일 (GCC 기준)

```bash
gcc treasure_game.c -o treasure_game.exe
```
### ✅ 2. 실행
```
treasure_game.exe
```

### 🎮 게임 규칙
```
■ 방향키로 캐릭터 이동
■ 보물을 찾으면 문제(덧셈) 1회 출제 → 정답 시 보물 획득
■ 오답 입력 또는 지뢰(※) 밟을 시 맵 초기화
■ 보물 2개 획득 시 게임 클리어
■ 클리어 후: r = 재시작, h = 기록조회, 기타키 = 종료
```

### 🏆 기록 저장 방식

- 최대 100명의 플레이 기록 저장 가능  
- 플레이어별 최대 100개의 기록 보존  
- 시간 기준 오름차순 정렬 후 출력  

### 출력 예시
```
<< 게임 기록 >>
1등: GON - 12.84초
2등: YJ - 18.53초
3등: MIN - 22.11초
```
